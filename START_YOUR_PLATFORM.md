## Quick Start (30 seconds)

### Step 1: Start the Platform
Double-click: **`start.bat`**

This will:
- ✅ Start backend API (port 5000)
- ✅ Start frontend app (port 3000)
- ✅ Open browser automatically

### Step 2: Verify Real Data
Visit: **`http://localhost:3000/chart/public-health-expenditure-trends`**

Look for these **real numbers from YOUR data**:

If you see these exact numbers → **✅ Everything is working!**

### Step 3: Explore
